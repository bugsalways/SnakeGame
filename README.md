# 贪吃蛇游戏

一个使用HTML、CSS和JavaScript开发的经典贪吃蛇游戏。

## 游戏特性

- 🎮 **经典玩法**：控制蛇移动，吃到食物增长得分，避免撞到墙壁或自身
- 🎯 **分数系统**：记录当前得分和历史最高分（使用localStorage保存）
- ⚡ **递增难度**：随着得分增加，蛇的移动速度会逐渐提高
- 🎨 **美观界面**：现代化设计，渐变背景和流畅动画
- 📱 **响应式布局**：适配桌面和移动设备
- 👆 **触摸屏支持**：移动设备上提供虚拟方向键
- ⌨️ **键盘控制**：支持方向键和WASD控制

## 游戏控制

### 桌面端
- **方向键**：控制蛇的移动方向
- **WASD**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **回车键**：开始游戏
- **按钮**：界面上的开始、暂停和重置按钮

### 移动设备
- **虚拟方向键**：点击屏幕上的方向按钮控制
- **按钮**：界面上的开始、暂停和重置按钮

## 如何运行

1. 将项目文件下载到本地
2. 双击 `index.html` 文件在浏览器中打开
3. 开始游戏并享受乐趣！

## 项目结构

- `index.html`：游戏的HTML结构
- `style.css`：游戏的样式和响应式设计
- `script.js`：游戏的核心逻辑和交互功能
- `README.md`：项目说明文档

## 技术栈

- **HTML5**：用于创建游戏界面结构
- **CSS3**：用于样式设计和响应式布局
- **JavaScript**：用于实现游戏逻辑和交互
- **Canvas API**：用于游戏渲染

## 游戏规则

1. 使用方向键或WASD控制蛇的移动方向
2. 吃到食物后，蛇的身体会变长，同时得分增加
3. 撞到墙壁或蛇自身的身体会导致游戏结束
4. 随着得分增加，蛇的移动速度会逐渐提高，增加游戏难度
5. 游戏最高分将保存在本地，下次打开游戏时仍然可见

## 开发说明

游戏使用Canvas进行渲染，通过JavaScript实现游戏循环、碰撞检测和用户交互。项目采用模块化设计，代码结构清晰，便于维护和扩展。
